---
title: "Getting started"
---

La mejor manera de empezar con React Three Fiber es instalarlo usando npm o yarn:

```bash
# Usando Yarn:
yarn add react-three-fiber three

# Usando NPM:
npm i react-three-fiber three
```

Tambien puedes usarlo desde un CDN:

```html
<script src="https://unpkg.com/react-three-fiber@4.2.10/web.cjs.js"></script>
<script src="https://unpkg.com/three@0.118.3/build/three.js"></script>
```
