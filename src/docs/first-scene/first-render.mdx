---
title: Creando nuestro primer cubo
---

Vamos a crear nuestro primer cubo usando `react-three-fiber`.
Lo primero que tenemos que hacer es importar `Canvas` desde `react-three-fiber` y luego envolver el padre de la escena 3D `<Sene />` con él.

También puedes envolver toda tu aplicación así:

```jsx
import ReactDOM from "react-dom";
import React from "react";
import { Canvas } from "react-three-fiber";

ReactDOM.render(
  <Canvas>
    <Scene />
  </Canvas>,
  document.getElementById("root")
);
```

Ya tenemos todo configurado para crear nuestra escena y agregarle nuestro primer cubo.

Para mostrar cualquier tipo de objeto en nuestra escena necesitamos crear un `mesh` (malla), y esto consiste en:

- Una geometría, en este caso nuestro cubo, pero puede ser cualquier geometría de [threejs](https://threejs.org/docs/#api/en/geometries/BoxBufferGeometry) o incluso un modelo propio hecho en [Blender](https://www.blender.org/) o algún otro software.
- Un material, este le dice a la malla como se debe pintar este objeto. En nuestro caso solo es un simple color que reacciona a la luz.

Ahora podemos crear estos componentes y envolverlos en una malla así: 

```jsx
function Scene() {
  return (
    <mesh>
      <boxGeometry attach="geometry" args={[1, 1, 1]} />
      <meshStandardMaterial attach="material" color="#6be092" />
    </mesh>
  );
}
```

Tenemos una geometría pero tiene dos problemas, es negra y parece 2D ya que solo podemos ver un lado.

Vamos a rotarlo un poco para que podamos verlo mejor: 

```jsx
function Scene() {
  return (
    <mesh rotation={[0, 10, 0]}>
      <boxGeometry attach="geometry" args={[1, 1, 1]} />
      <meshStandardMaterial attach="material" color={"#6be092"} />
    </mesh>
  );
}
```
