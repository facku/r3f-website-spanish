---
title: Agregando Luces
---

Una escena 3D será negra a menos que haya luz y en el mundo de threejs hay muchos tipos de luces. Los más comunes son:

- AmbientLight (Luz ambiental): esta es una luz que está en todas partes, no brilla en ningún lugar en particular, sino que se extiende por toda la escena.

- Point Light (punto de luz): como su nombre lo dice, una luz que se enfoca en un punto, puedes pensar en ella como una lámpara.

Comencemos agregando un `ambientLight` en nuestra escena:

```jsx
function Scene() {
  return (
    <>
      <ambientLight />
      <mesh rotation={[0, 10, 0]}>
        <boxGeometry attach="geometry" args={[1, 1, 1]} />
        <meshStandardMaterial attach="material" color={"#6be092"} />
      </mesh>
    </>
  );
}
```

Nuestro cubo ahora es verde, como puede ver, pero dado que agregamos una luz ambiental, todos los lados están iluminados de la misma manera y parece demasiado consistente y nada relaista.

Agreguemos también un punto de luz para mejorarlo: 

```jsx
function Scene() {
  return (
    <>
      <ambientLight />
      <pointLight position={[10, 10, 10]} />
      <mesh rotation={[0, 10, 0]}>
        <boxGeometry attach="geometry" args={[1, 1, 1]} />
        <meshStandardMaterial attach="material" color={"#6be092"} />
      </mesh>
    </>
  );
}
```

Podemos ver que ahora un lado es mucho más brillante que el otro porque la luz incide en el cubo desde el lado derecho, ya que está posicionado "10" en el eje X y nuestro cubo por defecto se colocará en la posición "0" en los tres. eje.

Podemos verlo aún mejor si eliminamos el `ambientLight` y aumentamos la intensidad y movemos nuestra luz a `0` en el eje X así: 

```jsx
<pointLight position={[0, 20, 10]} intensity={1.5} />
```
